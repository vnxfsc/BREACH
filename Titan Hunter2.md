# BREACH - Titan Hunter  
游戏玩法与代币经济投资分析报告

---

## 执行摘要

BREACH 是一款 Solana AR 链游，玩家在真实世界 POI（如东京塔、涩谷十字路口）追踪并捕获来自异次元的巨型泰坦（Titans），灵感源自《环太平洋》与《哥斯拉》。  

**游戏差异化**：  
- 非可爱宠物，而是黑暗科幻巨兽美学  
- 位置优先（POI 固定生成，非玩家随机刷怪）  
- 真实探索 + 社交裂变（附近玩家可见、公会 Boss）  

**代币经济核心**：  
- 总量 10 亿 $BREACH  
- 捕获/升级/市场手续费 50–100% 销毁  
- F2P 每日保底捕获，付费买便利不买战力  
- 目标：每日 burn ≥ mint，实现长期通缩  

**投资价值判断**：  
玩法闭环稳健 + 经济模型健康，有望复制 Pokémon GO（AR 病毒传播）× Axie Infinity（Web3 经济）的路径。Beta 测试已具备落地基础。

## 1. 游戏玩法深度分析

### 核心闭环（5 步）
1. **狩猎**：App 显示附近 Breach 信号，固定生成于 POI  
   - POI 权重：地标 5.0、公园 3.0、商场 2.0、住宅 0.5  
   - 元素与地形强匹配（水域 70% 深渊系、山区 60% 火山系）  

2. **捕获**：节奏类“神经链接”小游戏（类似 osu! / beatmania lite）  
   - Class I 简单 → Class V 极高难度 + 5000 $BREACH 捕获费  
   - 反作弊：GPS 精度 + 速度阈值 + 多设备检测  

3. **养成**：6 位基因序列决定属性上限  
   - 升级/进化/融合消耗 $BREACH（100% burn）  
   - 融合：同元素 Lv20+ 泰坦合成更高潜力后代  

4. **战斗**：
   - PvE：自动挂机刷资源（保底留存）  
   - PvP：回合制 + 预测博弈（石头剪刀布深度版）  
   - 世界 Boss：周末合作 raid（例：东京塔 1000 万 HP，按伤害分 $BREACH + 稀有材料）  

5. **交易与社交**：链上市场 + 公会 + 附近玩家可见  

### 玩家分层与留存设计
| 玩家类型 | 占比估算 | 核心动机                  | 游戏内保障                     |
|----------|----------|---------------------------|--------------------------------|
| F2P      | ~80%     | 探索 + 收集乐趣           | 每日保底 1 次捕获，时间换资源  |
| 轻付费   | ~15%     | 便利 + 外观               | 传送券、背包扩容、皮肤（非 P2W）|
| 硬核     | ~5%      | 排行 + 深度策略           | 排行榜、公会战、Apex 首捕奖励  |

**差异化记忆点**：  
- “巨兽降临现实街头”的沉浸感  
- 全球 POI 统一权重 → 新玩家/不同时区公平  
- 东京实地测试优势（高密度 POI + 玩家基数）

**潜在风险与优化方向**：  
- GPS 漂移 / 室内信号弱 → 虚拟捕获券 fallback  
- 高阶泰坦太贵 → 活动碎片拼图保底路径

## 2. $BREACH 代币经济模型分析

### 基本参数
- 总量：1,000,000,000（10 亿）  
- 小数位：9  
- Devnet 合约地址：`CSH2Vz4MbgTLzB9SYJ7gBwNsyu7nKpbvEJzKQLgmmjt4`  
- 初始流通：5%（流动性 + 早期奖励）

### 分配结构（通缩导向）
| 类别         | 比例 | 数量     | 释放/归锁机制              |
|--------------|------|----------|----------------------------|
| Play-to-Earn | 35%  | 3.5 亿   | 4 年线性释放               |
| 生态系统     | 25%  | 2.5 亿   | DAO 治理                   |
| 团队         | 15%  | 1.5 亿   | 2 年 cliff + 2 年线性      |
| 国库         | 10%  | 1 亿     | 治理控制                   |
| 流动性       | 10%  | 1 亿     | TGE 提供 DEX 池            |
| 顾问         | 5%   | 0.5 亿   | 1 年 cliff                 |

### 通缩机制（核心竞争力）
| 行为             | $BREACH 消耗方式          | Burn 比例 | 备注                          |
|------------------|---------------------------|-----------|-------------------------------|
| 捕获（Class III）| 支付费用                  | 50%       | 越高阶 burn 越多              |
| 升级 / 进化      | 消耗费用                  | 100%      | 全额销毁                      |
| 融合             | 消耗两只泰坦 + 费用       | 100%      | NFT + Token 双重销毁          |
| 市场交易手续费   | 5% 手续费                 | 50%       | 剩余进国库                    |

**每日 Token 流目标**（DAU 10 万模拟）：
- 新 mint（P2E 奖励）：≈ 100,000  
- burn（捕获 + 升级 + 市场）：≈ 150,000–200,000  
- 净通缩：每日 5–10 万枚销毁  

**经济健康熔断机制**：
- mint > burn × 2 → 暂停新泰坦生成  
- 价格 24h 跌幅 > 50% → 国库回购 + 临时增 burn 活动  
- NFT 地板价崩盘 → 限时减少 Class IV/V 生成率  

### 玩家经济平衡
- F2P：靠时间（探索 + 日常）可持续玩  
- 付费玩家：贡献主要手续费与 burn，推动通缩  
- 硬核玩家：通过排行 / Boss 首杀获得高额奖励，形成内容生产者  

**结论**：  
经济模型偏保守、通缩压力强，适合长期持有。相比多数链游“无限挖矿 → 通胀崩盘”，BREACH 的 burn 设计更接近可持续路径。
